# Security

## Reporting Malicious or Incorrect Entries

This registry contains metadata about onchain contracts. If you find an entry that:

- Points to a malicious or fraudulent contract
- Contains incorrect or misleading information
- Impersonates a legitimate project

Please open an issue with the label "security" or contact the maintainers directly.

## Scope

This repository does not contain smart contract source code. Entries are metadata references only. The registry does not endorse or audit any contract listed within it.
